使う準備
　ScreenBlur_Lib::ScreenBlurを生成しなければならない。

宣言：　　void ScreenBlur_Lib::Init(const int t_winWidth, const int t_winHeight, const int t_alpha, const int t_offsetX1, const int t_offsetY1, const int t_offsetX2, const int t_offsetY2);

概要：　　ブラーの方向を初期設定する

引数：　　t_winWidth：ウィンドウサイズの横幅
　　　　　t_winHeight：ウィンドウサイズの縦幅
　　　　　t_alpha：0～255を指定する、その指定値まで透過しながら移動する（最大で127が丁度いい）
　　　　　t_offsetX1：右へのフレームの移動値
　　　　　t_offsetY1：下へのフレームの移動値
　　　　　t_offsetX2：右へだんだん伸びをするようになる
　　　　　t_offsetY2：下へだんだん伸びをするようになる
戻り値：　なし

解説：　　　ブラーの方向を設定します。
　　　　　　例えば、ScreenBlur_Lib::ScreenBlur m_screenBlurしたあとにすぐこの関数を一度だけ呼ばなければならない。





宣言：　　void ScreenBlur_Lib::ReplayInit();

概要：　　ブラー処理の描画に関するものを再度初期化する

引数：　　なし
戻り値：　なし

解説：　　　ブラー処理を行う直前に描画の内容を再度初期化しないといけないため、これを呼ぶ必要がある。





宣言：　　void ScreenBlur_Lib::Release();

概要：　　ブラー処理にて使っていたものを解放する

引数：　　なし
戻り値：　なし

解説：　　　ブラー処理にて裏画面用の画像として生成したものを削除する。





宣言：　　void ScreenBlur_Lib::PreRenderBlurScreen();

概要：　　ブラーのための準備をする

引数：　　なし
戻り値：　なし

解説：　　　ブラーのための準備を行う。
　　　　　　ブラーをする直前にこの関数を呼ぶようにする。
　　　　　※注意点として、この関数の前に画像を描画しても描画されない。





宣言：　　void ScreenBlur_Lib::PostRenderBlurScreen();

概要：　　ブラーの処理を行う

引数：　　なし
戻り値：　なし

解説：　　　ブラーの処理を行う。
　　　　　　PreRenderBlurScreen()との間にある画像のすべてをブラー処理させる。
