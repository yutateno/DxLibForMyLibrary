使う準備
　四角形なら四辺の情報を保持するColl2D_Lib::SBoxを作成しなければならない。
　円なら座標と半径の情報を保持するColl2D_Lib::SCircleを作成しなければならない。

宣言：　　bool Coll2D_Lib::CheckColl(const SBox& t_boxA, const SBox& t_boxB);

概要：　　四角形同士の当たり判定をする

引数：　　t_boxA：四角形の情報
　　　　　t_boxB：四角形の情報
戻り値：　false：当たっていない
　　　　　true：当たっている

解説：　　　四角形同士の当たり判定を調べる。





宣言：　　bool Coll2D_Lib::CheckColl(const SCircle& t_circleA, const SCircle& t_circleB);

概要：　　ブラー処理にて使っていたものを解放する

引数：　　t_circleA：円の情報
　　　　　t_circleB：円の情報
戻り値：　false：当たっていない
　　　　　true：当たっている

解説：　　　円同士の当たり判定を調べる。





宣言：　　bool Coll2D_Lib::CheckColl(const SBox& t_box, const SCircle& t_circle);

概要：　　ブラーのための準備をする

引数：　　t_box：四角形の情報
　　　　　t_circle：円の情報
戻り値：　false：当たっていない
　　　　　true：当たっている

解説：　　　四角形と円のあたり判定を調べる。
