宣言：　　void FileReadWrite_Lib::FileReaderWrite::Init();

概要：　　初期化する

引数：　　なし
戻り値：　なし

解説：　　　使う前にデータを削除する。





宣言：　　bool FileReadWrite_Lib::FileReaderWrite::FileCSVRead(const std::string t_file);

概要：　　CSVのファイルを読み込む

引数：　　t_file：読み込むファイル
戻り値：　false：読み込み失敗時
　　　　　true：読み込み成功時

解説：　　　ファイルを読み込む処理をさせる。





宣言：　　std::vector>& FileReadWrite_Lib::FileReaderWrite::GetMapData();

概要：　　読み込んだデータを出力する

引数：　　なし
戻り値：　ファイル内データ：（読み込めた場合の）ファイルの中身

解説：　　　FileCSVReadにて読み込んだファイルの中身を出力する。しかし読み込めていなかったら空が出力される。





宣言：　　bool FileReadWrite_Lib::FileReaderWrite::FileCSVWrite(const std::string t_file, const std::vector>& t_data);

概要：　　CSVのファイルを書き込む

引数：　　t_file：書き込むファイル
　　　　　t_data：ファイルに書き込むデータ
戻り値：　false：書き込みファイル作成失敗時
　　　　　true：書き込みファイル作成成功時

解説：　　　データをファイルに書き込む処理をする。
