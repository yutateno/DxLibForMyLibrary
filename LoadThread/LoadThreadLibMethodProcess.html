宣言：　　LoadThread::LoadThread(const int max, const std::vector path, const std::vector type, std::function t_loadFunc);

概要：　　コンストラクタ

引数：　　max：読み込みをするファイルの総数
　　　　　path：読み込みをするファイル名
　　　　　type：読み込みをするファイルの種類
　　　　　t_loadFunc：ロード画面の引数三つの関数（経過フレーム時間、ロード予定個数、ロード済み個数）
戻り値：　なし

解説：　　　コンストラクタ





宣言：　　void LoadThread::Process();

概要：　　ロード処理及びロード画面の描画

引数：　　なし
戻り値：　なし

解説：　　　ロード処理を行い、ロード画面の描画をしている。





宣言：　　const std::vector LoadThread::GetFile() const;

概要：　　ロードを終えたファイルを渡す

引数：　　なし
戻り値：　ロードファイルの中身

解説：　　　ロードしたファイルの中身を渡す。





宣言：　　const bool LoadThread::GetEnd() const;

概要：　　ロードを終えたかどうか

引数：　　なし
戻り値：　true：ロードを終えた
　　　　　false：ロード中
解説：　　　ロードを終えたかどうかを返す。
