宣言：　　void ScreenBlur_Lib::Process(const int max, const std::vector<　std::string> path, const std::vector<　ELOADFILE> type);

概要：　　非同期処理を行うプロセス、また描画も担っている

引数：　　max：読み込みをするファイルの総数
　　　　　path：読み込みをするファイル名
　　　　　type：読み込みをするファイルの種類
戻り値：　なし

解説：　　　非同期処理を行う。そのさい描画もこの関数が担っている。





宣言：　　const std::vector<　int> ScreenBlur_Lib::GetFile() const;

概要：　　ロードしたファイルの値を渡す

引数：　　なし
戻り値：　ロードしたファイルの値を返す

解説：　　　非同期で読み込んだファイルの値を渡す。





宣言：　　const bool ScreenBlur_Lib::GetEnd() const;

概要：　　ロードが終えたかどうかを渡す

引数：　　なし
戻り値：　true：ロード終了
　　　　　false：ロード中

解説：　　　ロードが終えたかどうかを渡す。ロード中を把握できるように思えるが、出来ない。





宣言：　　void SetLoadFunc(std::function<　void(const int, const int)> t_loadFunc);

概要：　　ロードの画面用の関数を受け取る

引数：　　t_loadFunc：ロード画面の関数ポインタ
戻り値：　なし

解説：　　　ロード中の画面の関数を渡す。
